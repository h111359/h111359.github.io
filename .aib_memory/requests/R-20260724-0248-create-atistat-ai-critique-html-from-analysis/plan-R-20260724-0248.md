## Goal

Create `atistat/ai-critique.html` as a standalone static Bulgarian-language HTML page presenting the ATISTAT site quality audit findings for the site owner. The page reuses the ATISTAT theme CSS (`atistat/wp-content/themes/atistat/assets/css/main.css`) and header/footer HTML from `atistat/index.html`. It contains three content sections: (1) Executive Summary, (2) Pros/Cons, and (3) Improvement Suggestions with English AIB brain prompts per task. No modifications to any existing atistat pages. The source material is the archived audit at `.aib_memory/requests/R-20260723-2300-analyze-the-site-in-folder-atistat/analysis-R-20260723-2300.md`; this path must not appear in any page text.

Impacted components: `atistat/ai-critique.html` (new file), `.aib_memory/context.md` (documentation update).

## Constraints

- All changes confined to `atistat/` folder and `.aib_memory/` (context update only).
- Must operate as a static HTML file with no server-side code.
- Page text is Bulgarian; embedded improvement prompt blocks are English in AIB brain format (`## Input` / `## Goal`).
- No modification to `atistat/index.html`, `atistat/index-en.html`, or any other existing atistat page.
- Server-side issues (HTTPS redirect, HSTS, HTTP security headers, compression, caching) excluded from improvement tasks; may be noted as awareness items in prose.
- Section 3 covers only Critical and High priority issues fixable within `atistat/` file scope (approximately 8 tasks).
- No numeric scores anywhere on the page; qualitative narrative only.
- Page excluded from search engine indexing via `<meta name="robots" content="noindex, nofollow">`.
- Header nav replaced with a single "← Обратно към АТИСТАТ" link to `index.html`; no same-page anchors; no language switch element.

## Success criteria

- `atistat/ai-critique.html` exists and renders in a modern browser without console errors with all three content sections visible.
- All page text (except prompt blocks) is in Bulgarian.
- All prompt blocks use the AIB brain `## Input` / `## Goal` format in English.
- Each Section 3 task heading contains an inline Bulgarian priority label ([Важно] or [Критично]).
- `<head>` contains `<meta name="robots" content="noindex, nofollow">`.
- Header nav contains exactly one link — "← Обратно към АТИСТАТ" pointing to `index.html` — with no language switch.
- Visual theme matches `atistat/index.html` (same CSS, header SVG, and footer).
- `git diff --name-only` shows only `atistat/ai-critique.html` and `.aib_memory/` files changed.
- `context.md` updated with new Solution entry and four new Issues entries.

Task 1 heading: "[Важно] Оптимизация на зареждането на главното изображение"
Bulgarian description: Hero изображението се зарежда с атрибут lazy, което причинява забавяне от около 4 секунди при мобилни устройства. Замяната с eager зареждане и prioritizиране на ресурса значително ще подобри LCP показателя.
English AIB brain prompt:
```
## Input
In file `atistat/index.html`, the hero image in the `#hero` section currently has `loading="lazy"`. Change the attribute to `loading="eager"` and add `fetchpriority="high"` to the same `<img>` element. Do not change any other attributes or surrounding HTML.

## Goal

```

```
## Goal

```
