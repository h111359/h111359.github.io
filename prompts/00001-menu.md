Trigger: When I write wimple a request with only one word "menu"

## Process
1. Define what actions you know are defined
2. Respond with a numbered list with the actions (commands) with action name and short description
3. Instruct the user to choose a number
4. Number 0 is reserved for menu command
5. Once I select a command by typing its corresponding number, you will execute that command and provide me with the results or further instructions as needed.

## Objectives

- MUST NOT invent menu entries. If no evidence - do not include menu item
- MUST contain only defined already commands with explained behavior
- MUST NOT guess or add more commands 
- MUST NOT change the meaning of the commands beyond the defined already

## Example

MENU
-----

0. menu: shows the current menu
1. explain: provide explanation of the last answer
